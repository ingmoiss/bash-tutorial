#Bash Tutorial
----------
**- Make make the file executable**
Run `chmod +x hello_world.sh`

**- Execute bash script**
Run `./hello_world.sh`

**- Create local variable with:**
`local`

**- Bash Comparations**
|  Bash | Operator  |
| :------------: | :------------: |
| -lt  | <  |
| -gt  | >  |
| -le  | <=  |
| -ge  | >=  |
| -eq  |  == |
| -ne  |  != |

**- String Comparisons**

|  Bash | Operator  |
| :------------: | :------------: |
| =  | equal  |
| !=  | not equal  |
| <  | less then  |
| >  | greater then  |
| -n s1  |  string s1 is not empty |
| -z s1  |  string s1 is empty |
